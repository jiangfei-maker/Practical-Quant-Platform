## 5. 数据源管理策略

### 5.1 稳定性保障
*   **多源互备**: 核心财务数据同时接入 AkShare (主) 和 Tushare (备，如有Token)，或直接爬取单页作为兜底。
*   **本地缓存**: 所有历史数据第一时间持久化到 DuckDB/Parquet，减少重复请求。

### 5.2 合规与反爬
*   **请求限速**: 严格控制爬虫请求间隔 (Random Sleep 1-3s)。
*   **代理池**: 接入免费/付费代理池，避免 IP 被封禁。
*   **User-Agent 池**: 随机切换浏览器指纹。
